# Groupe de boulei_n

## Prerequisite

1.[install elm](https://guide.elm-lang.org/install/elm.html)
2. Download the index.html file

## Installation
````
cd C:\Users\<username>\
mkdir project
cd ./project
elm init
````

## Build
````
elm make src/Main.elm --output elm.js
````

## Launch
````
elm reactor
````
